# Custom Resume Website

A modern, responsive resume website built with React and Vite.

## Features

- Clean and professional design
- Responsive layout (mobile-friendly)
- Easy to customize with your own information
- Component-based structure for easy maintenance
- Print-friendly styles

## Getting Started

### Installation

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` folder.

### Preview Production Build

```bash
npm run preview
```

## Customization

To customize the resume with your own information, edit the following component files in the `src/components/` directory:

- `Header.jsx` - Your name, title, contact info, and social links
- `About.jsx` - Your professional summary
- `Experience.jsx` - Your work experience
- `Education.jsx` - Your educational background
- `Skills.jsx` - Your technical skills and competencies
- `Contact.jsx` - Contact information and message

## Technologies Used

- React 18
- Vite
- CSS3
